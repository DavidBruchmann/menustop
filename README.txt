Menu Stop
=========

This is a TYPO3 extension.  
Allows to end main menu, sitemap and click path at each location.  
Each page can be the last visible or the first hidden one. To be
configured in "page properties". It is an extended alternative to
the page property checkbox "hide in menu".  

Manual
------

The manual is resided in the folder "doc".  
Depending on the TYPO3 version and other extensions it might be
possible to read or download the manual in the backend.
